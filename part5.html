Part 5: Mini Project - Bouncing Ball

Task 11: Interactive Bouncing Ball Animation

Watch the ball move across the screen and bounce back!

Project Features:

  • Start Animation: Ball moves across screen horizontally and bounces back
  • Bounce Vertical: Ball bounces up and down
  • Bounce Diagonal: Ball moves diagonally and bounces
  • Continuous Bounce: Ball keeps bouncing in a loop
  • Stop: Stops all animations
  • Reset: Returns ball to starting position
